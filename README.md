# Zoomify - Video Conferencing App

A real-time video conferencing app built with React (front-end) and Node.js (back-end).

---

## Live Demo
Check out the live version:

[![Vercel](https://img.shields.io/badge/Live-Vercel-brightgreen)](https://zoomify-ruddy.vercel.app)  


---

## Features
- Real-time video and audio calls
- Create or join rooms instantly
- Responsive UI
- Chat functionality
- Multiple participants support

---

## Tech Stack
- Frontend: React, Bootstrap
- Backend: Node.js, Express
- Real-time communication: Socket.io
- Deployment: Vercel (Frontend), Render/Other (Backend)
